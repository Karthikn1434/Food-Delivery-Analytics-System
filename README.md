# Food Delivery Analytics System

## Project Overview

This project analyzes food delivery order data using Python, Pandas, NumPy, SQL, and Matplotlib. The goal is to understand customer behavior, restaurant performance, revenue trends, ratings, and delivery efficiency.

## Tools Used

* Python
* Pandas
* NumPy
* SQLite (SQL)
* Matplotlib
* Google Colab

## Project Workflow

* Dataset Understanding
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Data Visualization
* NumPy Analysis
* SQL Analysis
* Business Insights

## Key Findings

* Pizza Hut generated the highest revenue.
* Mumbai generated the highest revenue.
* Helen was the highest-spending customer.
* Average rating was 4.375.
* Average delivery time was 31.875 minutes.
